# CSP10 App 🚀

> Just an app for the best Cyber Security Professionals in the world.

Welcome to the **CSP10 App** – a modern Flutter application designed to serve as the digital home for our group.  
This app brings together everything we need for our educational journey: collecting quotes and a drinking game.

## ✨ Features

### 📊 Drinking game
- Send virtual bears to others
- If you receive a bear, you've got five minutes to empty your current drink
- Use your bears wisely!

### 📚 Collection of quotes
- Track funny quotes
- Add or delete your own quotes

## 🛠️ Tech Stack

### Core Technologies
- **Flutter** – Cross-platform UI framework
- **Dart** – Programming language

### Separate Backend
- **FastAPI** - Web framework for building APIs with Python
